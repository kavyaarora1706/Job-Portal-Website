# Worklyn

A modern job portal connecting talented professionals with top startups and tech giants.

## Overview

Worklyn is a sleek, employee-focused job portal designed to help job seekers find their dream positions at leading companies. Built with a modern dark theme and intuitive user interface, Worklyn makes job hunting a seamless experience.

## Features

### For Job Seekers

- **Smart Job Search**: Search by job title, keywords, and location
- **Trending Companies**: Quick access to jobs at top companies (Reddit, Spotify, Zepto, Airbnb, and more)
- **User Profiles**: Create and manage comprehensive professional profiles
- **Resume Management**: Upload and store resumes in multiple formats (PDF, DOC, DOCX)
- **Skills Showcase**: Add and display your technical skills with an intuitive tag system
- **Secure Authentication**: Login and signup functionality with password confirmation

### User Interface

- **Modern Dark Theme**: Eye-friendly dark interface with cyan/blue accents
- **Smooth Animations**: Elegant wave patterns and interactive elements
- **Intuitive Navigation**: Clean, straightforward user flow

## Pages

1. **Landing Page** (`index.html`)
   - Hero section with tagline
   - "Let's Start" call-to-action

2. **Job Search** (`job.html`)
   - Search bar with job title and location filters
   - Featured company showcase
   - Browse trending startups

3. **Authentication** (`login.html`)
   - Tabbed login/signup interface
   - Form validation
   - Password confirmation

4. **User Profile** (`profile.html`)
   - Personal information management
   - Professional bio section
   - Skills management with add/remove functionality
   - Resume upload with drag-and-drop support
   - Profile picture with edit capability

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with modern design principles and spline.
- **Icons**: Integrated icon system for enhanced UX
- **File Support**: PDF, DOC, DOCX (max 5MB for resumes)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/worklyn.git
   ```

2. Navigate to the project directory:
   ```bash
   cd worklyn
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 5501
   
   # Using Node.js
   npx serve
   ```

4. Access the application at `http://127.0.0.1:5501`
